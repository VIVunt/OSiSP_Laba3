1)  Папки:	
	DLL - dll библиотека
	DLL_Inject программа, которая внедряет dll в указанный процес
	DynamicImport	программа с динамическим подключение dll библиотеки
	fasmw17327 компилятор fasm
	StaticImport	программа со статическим подключение dll библиотеки
	TestProject тестовая программа для внедрения в неё dll библиотеки






2)  Для того, чтобв внедрить dll через дистанционный поток нужно:
	-Запустить программу "Project.exe" в папке "TestProject". Восле этого выскачи табличка с надписью "wait"
	-Запустить программу "ProcessExplorer" в SysInternalsSuite, найти процесс "Project.exe" и запомнить его PID
	-Запустить программу "Project.exe" в папке DLL_Inject через командную строку с передачей параметра PID процесса "Project.exe" из папки "TestProject". 
	-Закрыть окно с надписью "wait". Далее выскочит окно с надписью "new String".
	-Если за пустить программу "Project.exe" из папки "TestProject" без внедрения dll, то после "wait" выскочит окно с надписью "Source string".

3)  Строки
	Source string - исходная строка
	New string - заменённая строка
